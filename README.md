# Software Process Models Analysis and Case Study

**Course:** Software Engineering  
**Author:** Maria Iqbal  
**Semester:** 1  


## Summary
This assignment analyzes different **software process models** including Waterfall, Agile, Incremental, and Spiral models. It discusses their **advantages, disadvantages, and suitability** for various software development scenarios. The analysis is applied to a **realistic case study** to evaluate which models are most effective in practice.

## Files
- **Assignment_1_Software_Engineering.pdf** – Complete individual assignment report  
- **Diagrams/** – Optional folder containing flowcharts, diagrams, and illustrations used in the case study  


## Key Takeaways
- Understanding of various software process models  
- Comparison of their effectiveness in real-world software projects  
- Practical application of models in a case study scenario  
- Insights on selecting the right model based on project requirements
